Aviso de privacidad

¿Qué tipo de información recopilamos?

Recopilamos tu correo electrónico (email) y contraseña (cifrada) cuando usas nuestro Producto.
Por ejemplo, cuando te registras para crear una cuenta. Guardamos tus datos para tener un control de los usuarios, poder generar un certificado para cada uno y asignarles estos certificados a cada uno de nuestros usuarios registrados.
No recopilamos información personal del usuario como nombre, edad, sexo, religión, etc. Puesto que no es necesaria esta información para el correcto funcionamiento de este prototipo.

Generación de certificado

El certificado no incluirá información del usuario, esta información es proporcionada por otro componente externo a esta extensión. El usuario queda libre de exponer cualquier información en este certificado.
